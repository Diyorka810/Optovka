Веб-сервис для совместных покупок: Платформа, позволяющая пользователям объединяться для совершения оптовых покупок и получения скидок.

Форум лента объявлений (фильтруется по всем объявлениям, объявлениям созданым пользователем, объявлениям в которых он решил принять участие), чат к каджому объявлению, директ, общий чат (группы по сто человек), группой чат, личный кабинет

Объявление заполняется по форме: тип товара, модель, количество необходиме для получения скидки, количество товара которое он может себе позволить, личный адресс, адресс поставщика (мнимальное количесвто товара для участия в объявлении)

Отдельная сущность Поставщик 
Функционал форума для него никак не меняется
Но он имеет свое объявление о продаже

Объявления поставщика
Отличае от объявления юзера только в заголовке объявления (ну и он не указывает какое количество он готов приобрести)

Основная сущность Юзер
Юзер может добавить тему на форум, в которой он должен будет заполнить форму: какой тип товара его интересует, модель самого товара, количество товара необходимое для получение скидки, количество товара которое он может себе позволить купить, примерный адресс его проживания куда потом все приедут чтоб забрать свою часть и адресс оптового поставщика. (так же можно добавить фичу, которая будет ограничивать минимальное количество заказа от другого юзера)
Юзер может просматривать темы в ленте, либо находить их при помощи умного поиска.
Юзер может указать в теме что он тоже собирается купить данный товар, отметив количество которое он может себе позволить приобрести. если создателя темы не устраивает количество товара которое указал другой юзер он может удалить его из списка желающих, но он должен оставить комментарий с пояснением причины покоторой он его удалил (иначе бан)
Когда тема набирает нужное для скидки количество желающих купить товар она пропадает из ленты (либо становится серой) и умного поиска (поэтому и умный)
Юзер так же может оставить комментарий к теме чтобы получить больше информации или написать в личку создателю темы.

Оплату товара нужно производить переводом. (Не знаю как добавиьт защиту от кидал, мб просто при регистрации просить серию и номер пасспорта. ну и можно еще добавить галочку - провереный пользователь)

Склад количесвто товара

товар тип товара, модель, стоимость, время хранения

Не основная сущность Админ
Админ имеет право банить юзеров, закрывать темы или переность их в другой раздел.
Жалобы: юзер может пожаловаться на другого пользователя по причине : обмана, грубого общения, безпричинного удаление его из списка желающих купить товар

Модератор
